### Current Version
1.4

To run/build this yourself, you will need IOKit in your SDK (SDK's with IOKit can be found [here](https://github.com/CharlieWhile13/iOS-SDK-With-Passion)), Xcode and Theos installed.
Run the commands:
```
cd dra1n-app
pod install
cd ..
make package install
```
